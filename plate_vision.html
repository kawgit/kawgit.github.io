What it does

I developed Vision, a UCI (Universal Chess Interface) compatible chess engine capable of analyzing any given chess position to predict the best move and winning chances using a mini-max style search tree with iterative deepening and alpha-beta pruning.

How it works

For a given position, it starts with a search of depth 1, and records what it learns (e.g. winning chances or best moves) about each position it encounters in a hash table. It then searches with depth 2, using information from the hash table to speed up its search, for example by searching better moves before worse moves.

It efficiently computes legal moves for positions using bitboards, bit-manipulation, and magic hashing (hash functions made with magic numbers which are found through brute force, made to take advantage of hash collisions).

Currently, it uses a hand-crafted evaluation to evaluate the winning chances of positions, but will soon switch to a neural network based implementation.
