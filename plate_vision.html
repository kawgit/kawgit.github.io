Vision Chess

The above video shows a game played
by Vision Chess against itself with very
low time control.

I developed Vision, a chess engine written in C++ capable of analyzing any chess position to predict the best move and winning chances using a mini-max style search tree with iterative deepening and alpha-beta pruning.

For a given position, it will continuously explore the game tree recording, what it learns (winning chances and best moves) about each position it visits in a hash table. It explores the game tree selectively, prioritizing moves like checks, captures, promotions, or other moves that are heuristically likely to be best. Similarly, it spends little time considering moves that are likely to be bad.

It efficiently computes legal moves for positions using bitboards and magic hashing (hash functions with magic numbers made to take advantage of hash collisions).

To estimate the winning chances for any given position encountered in the search tree, it uses a neural network with a system that saves time by incrementally updating the features outputted from the first layer rather than re-computing them every time a move is done or undone.

I also made a simpler version Javascript with an HTML/CSS interface, here’s the link if you want to play: https://kawgit.github.io/vision-js.
